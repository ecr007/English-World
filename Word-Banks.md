- exactly == precisely
- nearly == almost
- over == more than | <> | under == less than
- increase == rise | <> | decrease == fall == drop
- A couple == Two
- Casual == Comfortable <> Formal == Conservative
- Unique == Unusual <> Common == Ordinary
